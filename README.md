## 📌 Project Overview

A hands-on Python project exploring **Object-Oriented Programming** concepts through real-world marketing analytics scenarios.

The notebook covers:
- Defining classes with `__init__` constructors and instance attributes
- 10 real-world OOP exercises modelling marketing operations
- Budget management, lead analysis, conversion tracking, and engagement scoring
- Conditional logic, method chaining, and clean class design patterns
